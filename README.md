# Employee Attendance System

A full-stack Employee Attendance Management System built using:

- Next.js (Frontend)
- NestJS (Backend)
- Firebase Firestore
- Firebase Admin SDK
- TypeScript

## Features

- Employee Registration
- Employee Login
- Punch In
- Punch Out
- Attendance History
- Live Dashboard
- Logout
- Firestore Database Integration

## Tech Stack

Frontend:
- Next.js
- React
- Tailwind CSS

Backend:
- NestJS
- Firebase Admin SDK

Database:
- Cloud Firestore

## Run

Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend

```bash
cd backend
npm install
npm run start:dev
```
